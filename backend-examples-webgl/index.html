Examples: WebGL back-end for MacroGL - fully optimized version

After having compiled and optimized properly the code for the application (using `sbt backendExamplesWebgl/fullOptJS` in the root folder), you should see the MacroGL example.

Basic square shape from triangles

You should see a pulsating square moving towards the left of the screen created by joining triangles on a red background.

Your browser doesn't appear to support the HTML5 canvas element.

Basic triangle

You should see a pulsating blue triangle on a red background.

Your browser doesn't appear to support the HTML5 canvas element.

Main banner

You should see the MacroGL homepage main banner animation.

Your browser doesn't appear to support the HTML5 canvas element.
